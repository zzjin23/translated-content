`touchcancel 觸控點發生失效的事件被觸發(例如太多觸控點時)`

## 一般資訊

- 規範
  - : [Touch Events](http://www.w3.org/TR/touch-events/#the-touchcancel-event)
- 介面
  - : TouchEvent
- 起泡事件

- 可取消
  - : 否
- 對象
  - : Document, Element
- 預設行為
  - : 無
