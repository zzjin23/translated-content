{{HTTPSidebar}}

HTTP **`205 Reset Content`** 狀態碼用來通知客戶端重置文件視圖，例如：清除表單內容、重置畫布狀態或刷新使用者界面。

## 狀態

```plain
205 Reset Content
```

{{Specifications}}

## 相容性事項

- 若是此回應在持久連線中錯誤地包含了主體，不同瀏覽器將會有不同的行為。細節參見 [204 No Content](/zh-TW/docs/Web/HTTP/Status/204) 。

## 參見

- {{HTTPStatus(204)}} No Content
