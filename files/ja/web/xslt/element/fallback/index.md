`<xsl:fallback>` 要素は指定された拡張子 (または、最終的には新しいバージョン) 要素がサポートされていない場合に使用するテンプレートを指定します。

### 構文

```
<xsl:fallback>
  TEMPLATE
</xsl:fallback>
```

### 必須属性

なし

### 任意属性

なし

### タイプ

命令は、テンプレート内に表示されます。

### 定義

XSLT, section 15

### Gecko のサポート

現時点ではサポートされていません。
