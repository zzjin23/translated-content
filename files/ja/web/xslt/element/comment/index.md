`<xsl:comment>` 要素は出力文書にコメントを書き込みます。テキストのみを含める必要があります。

### 構文

```
<xsl:comment>
  TEMPLATE
</xsl:comment>
```

### 必須属性

なし

### 任意属性

なし

### タイプ

命令は、テンプレート内に表示されます。

### 定義

XSLT, section 7.4.

### Gecko のサポート

サポート済み
