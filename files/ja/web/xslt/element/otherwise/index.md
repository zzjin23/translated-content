`<xsl:otherwise>` 要素は `<xsl:when>` 条件のいずれも適用されない場合に実行されるアクションを定義するために使用されます。他のプログラミング言語の `else` または `default` の場合と似ています。

### 構文

```
<xsl:otherwise>
  TEMPLATE
</xsl:otherwise>
```

### 必須属性

なし

### 任意属性

なし

### タイプ

サブ命令は `<xsl:choose>` 要素の最後の子としてテンプレート内に現れなければなりません。

### 定義

XSLT, section 9.2
