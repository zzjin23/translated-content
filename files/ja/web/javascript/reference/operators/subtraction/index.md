減算演算子 (`-`) は 2 つのオペランドの間で減算し、それらの差を生成します。

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

## 構文

```js
x - y
```

## 例

### 数値の減算

```js
5 - 3     // 2
3 - 5     // -2
```

### 非数の減算

```js
'foo' - 3 // NaN
```
