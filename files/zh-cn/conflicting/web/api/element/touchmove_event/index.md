{{APIRef}}

当触点在触控平面上移动时触发 `touchmove` 事件。

## 常规信息

- 规范
  - : [Touch Events](http://w3c.github.io/touch-events/#event-touchmove)
- 接口
  - : {{domxref("TouchEvent")}}
- 是否冒泡
  - : Yes
- 能否取消默认行为
  - : Yes
- 目标
  - : Document, Element
- 默认行为
  - : 无
