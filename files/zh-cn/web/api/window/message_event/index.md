**`message`** 事件是在窗口接受到消息（例如，从另一个浏览器上下文中调用 [`Window.postMessage()`](/zh-CN/docs/Web/API/Window/postMessage)）时，在 {{domxref('Window')}} 对象上触发的事件。

## 语法

```plain
window.addEventListener('message', function(event) { ... })
window.onmessage = function(event) { ... }
```
